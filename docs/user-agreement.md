2.1. Для исполнения настоящего Соглашения и обеспечения стабильной работы Сервиса Оператор оставляет за собой право обрабатывать следующие технические данные:

2.2. **Цели обработки:** Обработка технических данных, указанных в п. 2.1, осуществляется исключительно для защиты Сервиса от злоупотреблений (DDoS-атак, накруток, парсинга) и обеспечения равного доступа для всех Пользователей (Rate Limiting).
